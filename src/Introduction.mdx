# Le Jardin Radial

Component library for [Le Jardin Radial](https://jardin.pesce.cc) — a garden visualization tool that translates gardens into radial graphics revealing blooming cycles and color patterns throughout the year.

[Live app](https://jardin.pesce.cc) · [Source code](https://github.com/jpesce/le-jardin-radial)

## Architecture

Components follow **Atomic Design** principles:

- **Atoms** — Indivisible UI elements: Button, BackButton, Checkbox, Popover
- **Molecules** — Small compositions: MonthGrid, FlowerRow
- **Organisms** — Feature-level components: Reset, Share, SharedBanner, FlowerEditor, FlowerCatalog, FallbackPage, Footer
- **Assets** — Visual elements: Logo, SadFlower

## Language

Use the globe icon in the toolbar to switch between French and English. All components support both languages via the I18n provider.
